## 安装教程

安装SimMusic后，将此扩展拖入扩展页面或安装[扩展商店](https://github.com/PYLXU/pluginStore/)后直接安装

## 使用说明

1. 本扩展需要NCM支持扩展作为前置扩展，若您未安装，则启动软件时会提示您安装
2. 本扩展需要填写Api地址，设置->网易云Node JS API 扩展->API地址 填写NodeJS版本的API地址（若您没有接口，请自行搭建或打开瑞思音乐Web的网络切换夹）
